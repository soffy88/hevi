## 0. 定位【Δ】

生产线是**源无关引擎**。消费者 #1 = 中国全史视频库（产品线沿用 tongjian 之名，内容范围 = 全史）。上游"事典"角色由 **AII 历史 KU 接口**承担。

六级锁链映射不变：

```
Concept      ≈ EpisodePlan（事件序列 + 叙事框架）
Screenplay   ≈ 讲解稿 + 节拍
Design List  ≈ 地图 / 立牌 / 风格包资产解析
SceneStage   ≈ 桌面纸雕场（单一固定机位场）
Shot List    ≈ 编译镜头（含关键帧对）
Generation   ≈ 首尾帧插值
```

两条 spec 的对偶原则：

- 生产端：**结构只写一次，四处消费**（帧对 / 提示词 / 断言 / 贴点出自同一 VisualFact）
- 语料端：**抽取一次，四处消费**（事实在 KU 层确认一次，视频 / Mneme / 检索问答共用）

## 1. 全本约束与五条设计律

底料从通鉴单书扩为多书【Δ】，但量级逻辑不变：集数由**事件密度**决定，规划量级 600–1200 集起步；架构按全史建，**发布按断代打深**。换底料不换律：

| # | 律 | 内容 |
|---|---|---|
| L1 | **编译不手写** | 镜头 = 模板 × 参数。提示词、断言、贴点全部由编译器实例化，禁止逐集手写 |
| L2 | **注册表优先** | 勢力色、地名、人物、纪年、地图状态一律注册表化；【Δ】人名 / 地名 / 纪年 / 勢力注册表**主权在 AII 侧**（KU spec §3），生产端只读；勢力色与 MapState 仍归本侧 |
| L3 | **懒生长** | 资产库按需生产、缓存命中优先；复用率是一等指标 |
| L4 | **断言先行，贵前设门** | 每个模板自带断言集；关键帧门设在视频生成之前 |
| L5 | **人工有预算** | 人工分钟/集被测量；人只出现在 §2 的**五个**固定闸口【Δ】 |

## 5. 镜头语法库 S0–S12【Δ：新增 S12】

总原则不变：每镜头固定机位；景别变化 = 切镜；单镜 4–8s；VO 主时钟。

| ID | 名称 | 关键参数 | 首帧态 → 尾帧态 | 动效机制 | 断言重点 |
|---|---|---|---|---|---|
| S0 | 静帧延时 | duration | 同帧 | 装配级 hold | — |
| S1 | 全图铺陈 | map_state | 素纸灰调 → 色块到位 | 势力纸片依次滑入 | 色块计数、质心色 |
| S2 | 区域弹起 | region_id | 平铺 → 目标区折起、余区压暗 | 后缘折痕翻起 | 弹起区唯一性 |
| S3 | 撕裂/合并 | parent, children[] | 整块 → N 块沿裂线分离 | 沿预置裂线撕开 | 裂块数 = N |
| S4 | 扩张/收缩 | force, msA→msB | A 态 → B 态 | 新纸片叠入/揭走，禁 morph | 无 morph 违例 |
| S5 | 路线推进 | polyline, markers[] | 起点 → 全线+终点 | 纸带分段贴放，恒平贴 | 标记数 |
| S6 | 战役落点 | location, type | 无 → 标记+纸屑 | 标记片落下 | 标记数 = 1 |
| S7 | 人物立牌 | cutout_id, pos | 无 → 立牌立起 | 底折痕翻立 | 立牌身份分 |
| S8 | 数量对比 | items[] | 空 → 纸条柱状 | 纸条依次立起 | 纸条数、零文字 |
| S9 | 城池特写 | city_ms | 平铺 → 建筑弹起 | 多层 pop-up | 建筑层数 |
| S10 | 时间推移 | from, to, events[] | 游标 from → to | 纸带轴+游标 | 游标终位 |
| S11 | 翻页转场 | next_ms | 本页 → 下页 | 整页翻过 | 翻页后帧贴合 |
| **S12** | **对勘并置【新】** | conflict_ref, accounts[2] | 单版本平铺 → **双半幅并立**，各呈其说 | 页面沿中缝对折立起成双屏 | 半幅数 = 2、零文字 |

**S12 是内容差异化的签名镜头**：源标注（"《左传》·成公八年 / 《史记》·赵世家"）后期合成；两半幅各自的画面内容由各自 account 的摘述编译；presentation_hint 决定用 S12 还是"主线 + 角标注记"。史料对勘就是历史区的 engineering honesty——别的号讲一个圆滑的故事，我们把两块纸雕摆在一起。

模板六件套与升版纪律不变（参数 schema / 双态配方 / 锁模板 / 断言集 / SFX 贴点 / 失败模式；改动升版本记 decision_trail）。

## 9. 生成层（N7）【Δ 2026-07-21 G0 后修订：双后端】

模板注册表增字段 **`render_backend ∈ {deterministic_layers, i2v_keyframe_pair}`**：

- **`deterministic_layers`**（§5 分层揭示动画，`map_state.render_map_state_png` + 装配层）：
  **地图类模板默认走此后端**。零 provider、零色塌、B1a/B2 构建期恒绿（by construction）。
- **`i2v_keyframe_pair`**（keyframe_pair 模式，4–8s · 16:9 · 无镜头运动；重跑 ≤3；降级显式打标禁静默）：
  **可选后端，需逐模板证据授权**（provider 能力矩阵登记，见 `docs/specs/provider-capability-matrix.md`）。
  G0 实证：Wan-2.1 keyframe_pair 对 near_static 大色块可用、对**细分多色 transform 不可用**（塌分治色）——
  故地图分治镜**不授权** i2v。

**G0 结论（2026-07-21）**：G0 判 FAIL，失败分类 = **后端能力不适配**（i2v 不保小面积多色区），
处置 = **按模板换后端**（地图轨全部改 `deterministic_layers`）。探针使命完成，验收移交 **G0-D**。

## 10. QC 断言体系（N8）

程序化优先；VLM 只答极简可数问题。

B 组：B1 尾帧贴合 · B2 计数（期望值来自 locks.counts）· B3 中段质心色 · B4 抽帧 OCR · B5 材质违例问答。

**【Δ 2026-07-21 G0 后】B1 拆两层**：
- **B1a 结构达成（主判，程序化）**：对真实末帧在**矢量质心坐标**（`map_state.centroid_targets()`，坐标免费）
  采样比注册色 + 目标状态要素在位（分治块三色且分离 / 弹起区在位）。守场景状态不守像素。
- **B1b 全局 SSIM（护栏）**：**范围收窄为 `i2v_keyframe_pair` 模板专属**——`deterministic_layers` 路径
  B1a/B2 构建期恒绿，**B1b 不阻塞任何确定性路径**。i2v 模板的 B1b 阈值按模板类（near_static / transform）
  从实测分布定，**禁止从 n=3 拍**（须有合格样本）。
- **A2 = OCR**（非 edge_density 代理）；**A3 换 CLIP-ViT-B/32 后重标**（G0 用 pHash 代理）。
  「代理≠规格」为独立技术债，回填时按规格实现。

失败分类 → 处置：尾帧不贴 → 重跑→换时长→换 provider；标记多缺 → 数量锁强化→重跑；风格漂移 → 参考注入→重跑；机位漂移 → 首帧权重→重跑；**同类 ≥3 → 人工队列 + 模板缺陷单**（修模板不修镜头）。断言假阳性率抽样计量。
