## 1. 核心结论（先给判断）

1. **复用假设成立，且比原估更强。** 被证伪的是 naive txt2img 实现，不是「一次建场、多方位复用」。参数化几何 + 逐方位 depth 渲染才是该假设的真实现；原方案根本不存在可复用的「场」。
2. **成本集中在几何侧（B 轨），dressing 侧（C 轨）为薄层。** 这决定了整张复用链的价值分布：**重皮（换 dressing 保几何）的折扣比原估更深**，而**跨形制新建（换几何）的成本比原估更高**。
3. **复用面要再乘一个维度。** 原估分母是「服务期数 × 方位数」，实为「服务期数 × 方位数 × dressing 数」——几何跨 dressing 复用，D1/D2 共用同一套 boxes。
4. **两处必须上修的账**：A-HAN 重皮价值上修（几何近乎全复用）；A-SILK / A-SOUTH 跨季价值需按建筑形制相似度重判，**不可再按 A-HAN 的折扣率类推**。
5. **一处必须计入的新成本**：L1 身份锚从零重建（原「7/7 已建」为虚账，实物已灭失）。

## 2. 成本结构分层（据 CC 代码实证）

| 层 | 内容 | 量级 | 复用面 |
|---|---|---|---|
| **B · 几何** | 参数化 3D 底模（柱阵 EW×4、王座台、殿门框、矮屏，纯坐标 boxes）；虚拟相机环绕 + 逐方位 depth 导出；space_map→geometry 派生 | **L 级主体**，从零新建 | 跨 dressing × 跨方位 × 跨季重皮 |
| **C · dressing** | depth-ControlNet 变体接线、写实提示、per-dressing 提示（D1 素黑／D2 玄黑仪仗） | **小**（VRAM 已验、worker 分支已 scaffolded，增量仅 depth 变体接线） | per-dressing |
| **L1 · 身份锚** | 角色摄取 + 2D 锚 + Subject3D | 每具小（本地免费，占图像调用帽） | 跨场景 × 跨期（角色出现处即复用） |

**关键推论**：几何是昂贵且复用面最宽的层，dressing 是廉价且复用面最窄的层——这是最有利的成本结构。资产策略应当是**几何尽量少建、尽量宽用；dressing 放开建**。

## 3. 摊销重算

### 3.1 分摊公式

原式：`建场成本 / 服务期数`
新式：**`B几何成本 / (服务期数 × 方位数 × dressing数) + C成本 / (服务期数 × 方位数)`**

C 项分母不含 dressing 数（每套 dressing 各需自己的提示层），但 C 本身是薄层，主项由 B 决定。

### 3.2 A-QIN 几何的服务面

| 消费方 | 几何关系 | 吃 B 吗 | 吃 C 吗 |
|---|---|---|---|
| EP0 D1 王廷（前 227） | 原生 | ✓ | ✓ |
| EP0 D2 帝廷（前 221） | 同一大殿，换陈设 | 复用，0 增量 | ✓ |
| EP11 秦廷/秦吏场景 | 素秦风元素 | 复用 | ✓ |
| EP12 未央宫（A-HAN） | 汉承秦制，殿制结构相近 | **近乎全复用**（柱阵/台基/门框拓扑一致，尺度参数可调） | ✓ |
| EP13 未央宫廷议 | 同 EP12 | 复用 | 复用 |
| EP15 洛阳南宫 | 同型高台殿堂 | 复用 | ✓ |
| EP22 洛阳朝堂（北魏） | 同型殿堂 | 复用 | ✓ |

**A-QIN 的 B 几何实际服务 7 个 dressing 场景、跨 5 集**（原 MAP 记「服务 3 期 + 重皮 4 期」，量级一致但机制表述需改）。以每场景 8 方位计，B 的复用面 ≈ 7 × 8 = 56 次渲染消费。

**结论：B 的 L 级投资在秦汉宫殿这条线上是划算的**——它不是 EP0 的成本，是全季宫殿场景的共同基础设施。EP0 只承担其中一份。

## 4. 跨季投资重判（本文最重要的修正）

原 MAP 把 A-SILK（大漠丝路→七下唐）与 A-SOUTH（江南→七下宋）标为跨季投资，隐含假设是「建一次多季用」。在新成本结构下这个假设**部分失效**：

- **A-HAN 型复用（同形制换 dressing）**：几何全复用，折扣最深，**跨季价值上修**。宫殿线（秦→汉→魏晋→隋唐→宋）全部吃这一条，是全季最大的一笔几何红利。
- **A-SILK / A-SOUTH 型（异形制）**：大漠是开阔地景、江南是水田民居，与殿堂几何无共用。若强行走 B 轨，等于为每种形制各建一套参数化底模，**L 级投资 ÷ 少数几期 = 不划算**。

**策略（批准 2026-07-23）**：B 轨参数化几何**只投宫殿/室内结构化空间**——它复用面宽、透视一致性要求高、且是全季廷议签名场的载体。开阔地景（大漠、战场、江南水田、江面）走**低几何依赖路线**：这类场景本就少柱列少直角，视点一致性容错高，naive 生成 + 构图控制即可，不值得上 3D 管道。

这条策略把 SPEC-008 的 L 级投资从「全季资产基础设施」收窄为「殿堂类场景基础设施」，投资额不变而风险显著下降，且与 MAP-001 的期数分布吻合——宫殿场景占全季演绎期的多数。
