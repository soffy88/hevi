## 1. 三条贯穿线（这张表的骨架）

**L1 · 资产先行**：A-QIN（咸阳宫）是全季复用之王——直接服务 3 期，经「汉承秦制」重皮再服务 4 期以上。EP0 样片先建它：质量标杆与资产 ROI 双最优。

**L2 · 廷议签名场**：同一大殿资产四次换装，串起全季制度深水区——李斯议郡县（EP0）→ 主父偃献推恩（EP13）→ 外戚宦官博弈（EP15）→ 孝文帝汉化廷议（EP22）。「暂停—走入—拆潜台词」固定绑在这条线上，形成栏目记忆点。

**L3 · 每期一个置信示范点**：三级标注不是片尾字幕，是内容本体。每期预置一个「实录 vs 演绎」的对照瞬间（见 §2 表末列），全季信任资产靠它复利。

## 5. 全季固定结构与纪律（每期执行）

1. 结构：冷开场钩子 ≤60s → 旅人到场 → 主场（演绎/考古）→ 制度深讲（廷议线含暂停拆解）→ 现代视角收束 → 考点卡 ≤30s
2. 课堂可投屏纪律：考点卡之前零求关注、零网络梗，教师可整段直接投屏
3. 一主场 + 至多一对照切点，超出即拆期或砍
4. 三级置信标注全片常驻，每期一个预置示范点（§2 表末列）
5. 教材当索引：每期至少回答一个教材没空回答的「为什么」
6. 双减合规核查为 Mneme 导流的前置项（频道免费内容本身不受限）
