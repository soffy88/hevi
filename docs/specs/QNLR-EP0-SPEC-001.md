## 1. L1 Concept（锁定）

**一句话**：他赢了战争之后，真正的难题才开始——这一集讲清楚郡县制凭什么赢过分封制。

**三段叙事逻辑**：刺杀（统一是有人拿命阻止的事，公元前 227）→ 统一（六年后他坐在同一座大殿里，公元前 221）→ 治理（分封还是郡县：一场《史记》全文记录的真实辩论）。

**时间结构**：冷开场在前 227 年，正片主体在前 221 年，同一座咸阳宫大殿，两套 dressing 承载六年跨度——一个场吃整集，符合"一主场"纪律。

**本期回答的"教材没空回答的为什么"**：郡县制凭什么赢——不是嬴政拍脑袋，而是分封制刚刚给出过 800 年级别的失败样本（李斯原话：属疏远，相攻击如仇雠），且郡县提供了新机制：守/尉/监三权分置、流官任免、俸禄替代封地。

## 6. L6 Generation 技术备注

1. **多人镜**（S01/S02/S13/T01/T03/W 系）：compose→img2img 已验证路径，不训 LoRA。
2. **Subject3D**：6 具名角色全建（约 172s/个 CPU，合计约 17 min，忽略不计）。执行 2D/3D 共存原则：2D 管身份保真，3D 管视点自由——嬴政/荆轲跨 10+ 镜多方位，是 3D 锚的主要受益者。
3. **口型清单**：T02/T05/T07 共 3 镜，全部静态构图正/侧面——落在已验证能力的舒适区。
4. **高风险镜**：S07（环柱运动）唯一 P0 风险，Plan B 已备；S05 单独精修预算。
5. **生成顺序建议**：S01 最先跑（人数最多＝管线压力测试），S07 第二（风险最高早暴露），其余按序。
